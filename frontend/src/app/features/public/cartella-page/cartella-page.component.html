Odontogramma FDI integrato

Lo schema dentale in notazione FDI (Federazione Dentale Internazionale) mostra lo stato di ogni dente permanente a colpo d'occhio. Ogni elemento e' selezionabile per aggiornarne lo stato clinico.

  • Dente sano — nessun intervento registrato
  • Carie — trattamento necessario
  • Otturazione — otturazione composita o amalgama
  • Mancante — elemento non presente
  • Corona — elemento protesico
  • Impianto — fixture osteointegrata
